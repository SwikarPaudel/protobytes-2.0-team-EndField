## Implementation
1. Install `reqirements` from Terminal: `python install -r requirements.txt`. Folder is inside `{project_dir}/backend/`
2. Run server from `run.py`: `python run.py`
3. Run the frontend in `{project_dir}/frontend/` using [Godot Engine]([https://www.example.com](https://godotengine.org/)
